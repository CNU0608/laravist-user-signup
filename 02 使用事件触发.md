#使用事件触发

UserController.php
```
public function store(UserSignUpRequest $request){
    $data = [
        'name' => $request->get('name'),
        'email' => $request->get('email'),
        'password' => bcrypt($request->get('password'))
    ];

    User::register($data);

    return redirect('/success');
}
```
